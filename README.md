# Guzel - Art & Framing Studio

A high-end e-commerce platform for custom art framing, built with Next.js 14, TypeScript, Tailwind CSS, Shadcn UI, Framer Motion, and Zustand.

## Design System

- **Primary Background**: Deep Olive Green (#3e523f)
- **Accent**: Cream (#fdfcf6)
- **Text**: Dark Charcoal (#1e1e1e)
- **Typography**: 
  - Headings: Playfair Display (Serif)
  - UI Elements: Inter (Sans-serif)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app` - Next.js 14 App Router pages and layouts
- `/components` - React components
  - `/ui` - Shadcn UI components
- `/lib` - Utility functions
- `/public` - Static assets
